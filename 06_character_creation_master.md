# 6. キャラクリエイト / マスター編

## マスターの能力

マスターは以下の能力を保有しています。
1.  **令呪**: 各マスターにつき3画
2.  **汎用マスタースキル**: 下記の一覧から2つ習得
3.  **固有マスタースキル**: 下記の一覧から3つ習得

## 令呪

サーヴァントを従えるためにマスター全員に対し聖杯ないし監督役から配布される対サーヴァント用の絶対命令権。マスター一人につき最大3画配布され、「討伐令」の報酬として追加配布される場合もあります。

令呪にはスキル/宝具のリソースとして使用するほか、それ単体で自身のサーヴァントに対し以下の効果を発動可能です。

*   **① 判定支援(事前):** 判定直前に宣言。宣言後に行う判定に対し補正値『±8』。この補正値は他スキルの補正値と重複する。
*   **② 判定支援(事後):** 判定直後に宣言。直前に行った判定ダイスを振りなおし、振りなおす前と後のどちらを採用するか選択できる。
*   **③ 魔力供給:** 任意のタイミングで自身のサーヴァントのFPを「6」点即時回復させる。
*   **④ 絶対命令:** 特殊RP(自害など)を強制させる手段として使用する。内容によってはGMにより禁止されるので注意。

## 汎用マスタースキル

マスター全員が共通して保有しているスキル。下記の一覧から2つ選択し、習得します。下記のスキルを改造することは原則不可とします。

| スキル名 | 効果 |
| :--- | :--- |
| **魔力供給 I型** | **タイプ**: HP回復系<br>**発動**: クリンナップフェイズ開始時<br>**制限**: FP1点, 1シーン1回, 自身に対し使用不可<br>**効果**: 同盟含む味方サーヴァント1体のHPを「2」点回復。追加で自身のHPを消費することで、消費した点数分の回復量を上乗せ可能(上限:自身の【耐久】点, 最大+8)。 |
| **魔力供給 II型** | **タイプ**: FP回復系<br>**発動**: クリンナップフェイズ開始時<br>**制限**: FP1点, 1シーン1回, 自身に対し使用不可<br>**効果**: 同盟含む味方サーヴァント1体のFPを「2」点回復。追加で自身のFPを消費することで、消費した点数分の回復量を上乗せ可能(上限:自身の【魔力】点, 最大+5)。 |
| **回路賦活** | **タイプ**: 独自系<br>**発動**: クリンナップフェイズ開始時<br>**効果**: エンカウントフェイズにて戦闘が発生した場合でも自身のFP自然回復が発生する。ただし、回復量は通常時の半分(切上げ)となる。 |
| **回路連結** | **タイプ**: 独自系<br>**発動**: 任意<br>**制限**: FP1点, 1シーン【魔力】回<br>**効果**: 任意のタイミングで自身のサーヴァントが消費するFPを任意点だけ肩代わりする。(上限「自身の【魔力】点) ※発動コストFP1点とは別に、肩代わりするFPを消費する。 |
| **魔力装填** | **タイプ**: 独自系<br>**発動**: クリンナップフェイズ開始時<br>**制限**: FP2点, 1シーン1回<br>**効果**: 下記リストからアイテムを1つ入手。追加FP2点で更に1つ入手可能(1シーン最大2個)。アイテムは任意の対象に使用でき、所持上限は3つまで(譲渡不可)。<br>**<アイテム効果リスト>**<br>① 対象のHPを「3」点回復する。<br>② 対象のFPを「2」点回復する。 |

## 固有マスタースキル

それぞれのマスターが個別に保有する特殊能力。下記の一覧から選択または抽選を行い、習得します。
*   基本的に下記表の効果をそのまま使用。オリジナルや改造はGMの監修が必須。
*   各スキルは1キャラにつき1つまで。名前に「(抽選)」とあるものは、戦争中に1組しか保有できない。
*   強力なスキルはマスタースキル枠を2つ消費することもある（GM判断）。
*   マスターは、遭遇及びログ確保などで視認したサーヴァントのステータスを閲覧できる（スキル・宝具は別途真名看破が必要）。

| スキル名 | 効果 |
| :--- | :--- |
| **敵情視察** | **発動**: クリンナップフェイズ開始時<br>**制限**: FP1点, 1シーン1回, 未遭遇時限定<br>**効果**: 隣接エリアで発生した遭遇ログを1つ選択し入手できる。 |
| **事前調査** | **発動**: クリンナップフェイズ開始時<br>**制限**: FP1点, 1シーン1回<br>**効果**: マスター1名を指定し「探索判定」。成功時、対象マスターのキャラシートと個別チャット開設権を獲得。 |
| **隠蔽工作** | **発動**: クリンナップフェイズ開始時<br>**制限**: FP「発動回数×2」点, 自身の「知覚判定」成功<br>**効果**: 味方陣営の「魂喰らい」等による「事件/異変アナウンス」を1シーン後に遅らせる。 |
| **運転免許** | **発動**: ①逃走判定時 / ②エリア移動時<br>**制限**: FP2点, 自身の「知覚」or「肉体」判定成功<br>**効果**: ①蓄積されている逃走ペナルティを半減。②2つ先のエリアまで移動可能。 |
| **物資調達** | **発動**: クリンナップフェイズ開始時<br>**制限**: FP1点, 1シーン1回<br>**効果**: アイテムを1つ入手(追加FP2点で+1)。所持上限5つ。<br>**<アイテムリスト>**<br>①対象1体の攻撃or防御判定補正値『+2』(重複可)。<br>②対象1体の判定を振り直し(汎用アクション回数に含めず)。 |
| **強化触媒** | **発動**: キャラクター作成時<br>**効果**: サーヴァントの英雄点「+2」。 |
| **交渉巧者** | **発動**: 常時<br>**効果**: 個別チャット開設回数を「2回」から「4回」に増加させる。 |
| **乾坤一擲** | **発動**: 自身の[○]攻撃判定時<br>**制限**: シナリオ中1回<br>**効果**: 自身の最大FPを任意点減少させ、『減少させたFP×3』点の補正値を獲得(上限+12)。減少した最大FPは回復しない。 |
| **巻込まれ** | **発動**: キャラクター作成時/常時<br>**制限**: マスター初期英雄点「-1」, 特定スキルと同時習得不可<br>**効果**: 「雇われ人」の対象外となる。未遭遇の「事前調査」の初回を無効化。戦闘に参加したシーン終了時、自身の現在/最大FPが「1」点増加(最大5点まで累積)。 |
| **戦闘機動** | **発動**: 汎用アクション「配置転換」使用時<br>**効果**: 配置転換の使用回数を「1戦闘1回」から自陣営の探索技能「肉体/2(切上げ)」回に変更。 |
| **強化人間** | **発動**: キャラクター作成時<br>**制限**: 他の英雄点増強系スキルと同時習得不可<br>**効果**: マスターの英雄点「+5」(合計15点)。「神性」を除く強化属性を1つ獲得可。ステータスランク上限を「B(7)」に引き上げ。マスターへの即死効果を無効化。 |
| **伝承保菌** | **発動**: キャラクター作成時<br>**制限**: 「強化人間」「人外種族」「霊基融合」いずれかの習得が前提<br>**効果**: マスター自身にサーヴァントと同等の宝具(マスター宝具)を1つ持たせる(スキル枠1～2消費)。マスター宝具は1シーン1回、最低コストでのみ使用可能。 |
| **雇われ人(抽選)** | **発動**: キャラクター作成時<br>**効果**: 抽選で決まった参加者一人と同盟を締結した状態で戦争に参加できる。 |
| **土地管理(抽選)** | **発動**: キャラクター作成時<br>**制限**: シナリオ開始時、キャラシート全体公開<br>**効果**: スタート地点が霊地になる。霊地でのHP/FP回復量+1(上限5)。第一次クラス抽選の時点で開催地を指定可能。 |
| **現地在住(抽選)** | **発動**: キャラクター作成時/常時<br>**効果**: 「土地管理」保有者がいる場合、第一次クラス抽選時に開催地を知ることができる。いない場合は自身が開催地を指定可能。 |
| **魔術工房(抽選)** | **発動**: クリンナップフェイズ開始時<br>**制限**: シナリオ中1回, 1エリアに1つ<br>**効果**: エリアにオブジェクト「工房Lv.1」を設置。作成者不在時は成長しない。 |
| **副召喚主(抽選)** | **発動**: キャラクター作成時<br>**制限**: マスタースキル枠を2つ消費<br>**効果**: もう一体のマスター(サブマスター)を作成する。サブマスターは令呪を持たないが、戦闘に参加可能。 |
| **人外種族(抽選)** | **発動**: キャラクター作成時<br>**制限**: 同盟不可, 特定スキルと同時習得不可<br>**効果**: マスター英雄点「+10」(合計20点)。ステ上限「A(9)」。マスター即死無効。特定強化属性獲得。「魂喰らい」が任意発動可能になる。偶数シーンに最大HPが減少するデメリットあり。 |
| **霊基融合(抽選)** | **発動**: キャラクター作成時<br>**制限**: サーヴァント召喚不可, 特定スキルと同時習得不可<br>**効果**: マスター英雄点「+20」(合計30点)し、マスター自身をサーヴァントとして扱う。ステータス上限撤廃、クラス特性や宝具などもサーヴァント準拠で作成。 |
| **洗礼詠唱(抽選)** | **発動**: 自身の攻撃判定時<br>**制限**: 前衛時のみ, 「人外種族」保有者使用不可<br>**効果**: 手番を消費し、サーヴァント等に【魔力】or【幸運】に応じた防御無視ダメージを与える。「魂喰らい」実行者には追加ダメージ。 |

## 名前、職業、容姿の設定
マスタースキルが決まったら、その組み合わせに合ったマスターの名前、表向きの職業、容姿を考えます。この段階でキャラクターの立ち絵なども用意するとよいでしょう。

### ☆オプションルール: オープニングSS / 陣営画像 / ハンドアウト
GMは参加者の意志を確認し、下記の中から任意の創作物を採用します。
*   **オープニングSS**: 各陣営をイメージした前日譚や自己紹介SSを作成し、立ち絵と同時に公開する。
*   **陣営画像の作成**: サーヴァントとマスターの画像を用いて、陣営のイメージ画像を作成し、立ち絵と同時に公開する。
*   **ハンドアウトの作成**: マスターの設定や背景を記載した紹介文。各PLは開始時に他マスター1名のハンドアウトを獲得でき、交渉の材料などに使用できる。
