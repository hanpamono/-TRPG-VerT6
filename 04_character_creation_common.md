# 4. キャラクリエイト / 共通編

## 英雄点と探索点

キャラクリエイト時において各PCに与えられる基本リソースです。後述する能力値や探索技能への割り振り、スキル/宝具の習得リソースとして使用します。

| リソース名 | マスター | サーヴァント |
| :--- | :--- | :--- |
| 英雄点 | 10点 | 30点 |
| 探索点 | 15点 | 18点 |

## 能力値 (ステータス)

戦闘判定の基準値算出やリソースの計算に用いられる項目です。「英雄点」をそれぞれのステータスに振り分けることで各ステータスの「ランク」を決定します(※1)。

*   キャラクリエイト時、各ステータスには最低でも1点は振り分けねばならず、「0」にすることは原則不可。
*   特定の「スキル/宝具」等の効果により一時的に「0以下」にすることも原則不可だが、GMがバランスを鑑みて許可した場合に限り、一時的に「0以下」にしてもよい。その場合ランク表記は一律「E-」となる。
*   マスターの場合、各ステータスランクの上限は「C (5)」であり、それ以上にすることはできない。(※②)
*   遭遇か、あるいはスキル等によりログを獲得したマスターは、エンカウントフェイズ開始時や獲得したログで目撃したサーヴァントのステータスを閲覧することができる。この時、閲覧できるのはステータスのランクのみであり、スキルや宝具等のデータは後述する「真名看破判定」等を行う必要がある。

> ※①:英雄点を全ステータス均等に割り振った場合、マスターはオール「E+(2)」、サーヴァントはオール 「C+(6)」となる
> ※②:一部マスタースキルでその上限を変更することは可能。詳細はキャラクリエイト・マスター編参照

| 名称 | 概要 | 使用例 |
| :--- | :--- | :--- |
| **筋力** | 物理的な攻撃力を表す能力。 | バスター(B)判定、FP計算など |
| **耐久** | 肉体の強靭さを表す能力。 | HP計算 |
| **敏捷** | 身のこなしの速さを表す能力。 | クイック (Q)判定、C値計算、先制/逃走/追跡判定など |
| **魔力** | 身に宿る魔力の総量を表す能力。| アーツ(A)判定、FP計算、終了フェイズ時におけるFPの回復量など |
| **幸運** | 自身の「運命力」を表す能力。 | FP計算、C値計算、先制/逃走/追跡判定、奇襲防御判定など |

### ランク

対応する能力に割り振られた英雄点によってE～EXまで表示されます。尚、EXランクは1キャラにつき1つまでとします。
「A+(10)」以上のランクの必要英雄点は2点となり、端数については切捨て扱いとなります。
※英雄点を10点消費しても、ランクは「A+ (10)」にはならず、「A (9)」のままとなる。

| ランク | 実数値 | 必要英雄点 | RP目安 |
| :--- | :--- | :--- | :--- |
| E-   | 0      | ※          | 特定の条件下でのみ到達できるランク。原則としてこのランクの能力のキャラは作成できない。 |
| E    | 1      | 1点(合計1点) | 平均的な人間の能力 |
| E+   | 2      | 1点(合計2点) | |
| D    | 3      | 1点(合計3点) | その分野において一流を名乗れるレベル |
| D+   | 4      | 1点(合計4点) | |
| C    | 5      | 1点(合計5点) | 人体の限界値。一流の人間が外的な補助を受けて到達できるレベル |
| C+   | 6      | 1点(合計6点) | |
| B    | 7      | 1点(合計7点) | 神秘の領域。信仰による補正を受けて到達するレベル |
| B+   | 8      | 1点(合計8点) | |
| A    | 9      | 1点(合計9点) | 神話の世界でも一握りの者しか到達しえないレベル |
| A+   | 10     | 2点(合計11点) | |
| A++  | 11     | 2点(合計13点) | |
| EX   | 12     | 2点(合計15点) | 完全なる規格外。世界でも指折りの英雄しか辿り着けない領域。 |
| EX+  | 13以上 | --         | 特定条件でのみ到達できる、超越者の領域 |

## 必須属性と強化属性

キャラクリエイトの際、サーヴァントとマスターはいくつかの属性を設定しなければならない。属性にはそのキャラクターのパーソナリティを表す「必須属性」と、追加英雄点を得られる「強化属性」の2種類がある。

### 必須属性

「必須属性」には「性別」「アライメント」「天/地/人/星」の3つが存在する。

*   **性別**: 男性、女性、無性、性別不詳の4つから選択する。※無性、性別不詳を選択した場合は英雄点を1点消費する必要がある。
*   **アライメント**: 周囲に対する立ち位置である「秩序・中立・混沌」と、当人の気質を表す「善・中庸・悪」の2項目によって決まる属性。※バーサーカーの場合、例外的に「善・中庸・悪」に加えて「狂」というアライメントが追加されることもある。
*   **天/地/人/星**: 人類史においてどんな立ち位置にいたのかを表す属性。
    *   **天**: 神としての性質を持つ者。(カルナ、ギルガメッシュ等)
    *   **地**: 土着の伝承にて語られる人物。(アルトリア、フランケンシュタイン等)
    *   **人**: 史実の人物。現代に生きる人間も該当。(カエサル、織田信長など)
    *   **星**: 人類史の中で大きな希望を残した人物。(星の開拓者保有者など)

### 強化属性

「強化属性」には以下の8つが存在し、1キャラクターにつき最大で3つまで取得することができる。

| 属性名 | 追加英雄点 | 概要 |
| :--- | :--- | :--- |
| 神性 | 2 | 神に属する、あるいは神の血縁者としての側面を持つ英霊。(ヘラクレスなど) |
| 貴種 | 1 | 王や皇帝等、カリスマ的な存在と称された英霊。(カエサルなど) |
| 魔性 | 1 | 鬼や怪物などと呼ばれた逸話を持つ英霊。(アステリオスなど) |
| 竜種 | 1 | 竜としての力を宿す英霊。(アルトリアなど) |
| 巨大 | 1 | 巨人のように体躯が特別大きいとされる英霊(ダレイオスなど) |
| 獣性 | 1 | 獣としての特性を持つ英霊。(アタランテなど) |
| 妖精 | 1 | 伝承に伝わる妖精、あるいは精霊としての側面を持つ英霊(メリュジーヌなど) |
| 聖者 | 1 | 宗教的に徳の高い逸話を持つ英霊に付与される。(ゲオルギウスなど) |

*   「強化属性」は基本サーヴァントのみ取得可能で、マスターの場合はマスタースキル「強化人間」「人外種族」「霊基融合」いずれかの取得が必須となる。ただしマスター・サーヴァント問わず、相応の理由・設定・逸話が無ければGMが取得を却下することもあるので注意する。
*   「必須属性」と「強化属性」はエンカウントフェイズ開始時に遭遇した陣営やログを獲得した陣営が閲覧するステータス表に記載し、公開しなければならない。
*   各属性には対応する「特攻能力」が存在する(女性特攻、竜種特攻など)。この時、「必須属性」に対する「特攻能力」は「強化属性」に対する「特攻能力」より低めの倍率で設定される。

## 戦型(スタイル)とHP(ヒットポイント)/FP(フェイトポイント)

マスターとサーヴァントはキャラクリエイトの際に「戦型(スタイル)」を選択し、その戦型に応じた「HP(ヒットポイント)」と「FP(フェイトポイント)」という2つのリソースを算出する。

### ①戦型(スタイル)
そのキャラクターが得意とする戦い方を意味しており、以下の2つからイメージに近いものを1つ選択する。

| 戦型名 | キャラクター | HP計算式 | FP計算式 | 概要 |
| :--- | :--- | :--- | :--- | :--- |
| ストロング | マスター | 【耐久】×3+10 | 【筋力】+【魔力】+【幸運】 | 身体能力に優れたキャラクターが選択する戦型。<br>体力に優れる反面、魔力的リソースが少ない傾向がある。 |
| | サーヴァント | 【耐久】×3+20 | 【筋力】+【魔力】+【幸運】+2 | |
| テクニック | マスター | 【耐久】×3+6 | 【筋力】+【魔力】+【幸運】+2 | 魔力や技量に秀でたキャラクターが選択する戦型。<br>魔力的リソースが豊富な反面、耐久面にやや不安が残る。 |
| | サーヴァント | 【耐久】×3+12 | 【筋力】+【魔力】+【幸運】+6 | |

### ② HP (ヒットポイント)
*   キャラクターが持つ体力的リソース。ダメージなどを受けることで減少する。
*   このリソースが0以下になるとそのキャラクターは「気絶」状態へと移行し、さらにマイナス分がそのキャラクターの【耐久】×2」を超えるとそのキャラクターは「死亡/消滅」する。
*   「気絶」状態のキャラクターは行動不能となり、戦闘中であれば直前まで配置されていたフィールド(前衛/後衛)の中で放置される。また、戦闘中に「気絶」状態のキャラクターがHP1以上になった場合、次のラウンドから行動を再開できる。
*   「気絶」状態のキャラクターに対し、手番を消費して「止めを刺す」と他のキャラクターが宣言した場合、その宣言の対象となったキャラクターは「死亡/消滅」する。また、クリンナップフェイズが終了するまでの間にHP1以上にならないことが確定したキャラクターも、戦闘終了後「死亡/消滅」する。(※①、※②)

> ※①: エンカウントフェイズの段階でHP回復の見込みがないとGMが判断した場合、「死亡/消滅」処理を前倒しすることもある(任意)。
> ※②:スキル等の効果でHP0以下のまま行動しているキャラクターが聖杯戦争の勝者となった場合、「死亡/消滅」処理の前に聖杯を獲得する。

### ③FP (フェイトポイント)
*   キャラクターが持つ魔力的リソース。スキル/宝具等のコストとして使用することができる。
*   このリソースは0になっても特にペナルティを負うことはないが、いかなる手段であってもこのリソースが0未満になるような行動をすることは出来ない。
*   このリソースは戦闘が発生していないシーンのクリンナップフェイズ時に【魔力】/2(切上げ)」点回復する。
*   FPを消費する行動には、全キャラクター共通で行える「汎用アクション」が存在する。

#### 汎用アクション

| アクション名 | コスト | 効果 |
| :--- | :--- | :--- |
| **ブースト** | FP1点 or HP2点 | ・この効果は戦闘判定の直前に宣言する。<br>・任意の戦闘判定の補正値『+2』<br>・汎用アクション 「振り直し」との併用は不可 |
| **振り直し** | FP「その判定での使用回数」点<br>※1回目=1点, 2回目=2点 | ・この効果は戦闘/探索判定の直後に宣言する。<br>・直前に行った戦闘/探索判定ダイスを振り直す<br>・この効果は1つの判定につき2回まで使用できる。<br>・汎用アクション「ブースト」との併用は不可 |
| **スキルレジスト**| エンカウントフェイズ中:3点<br>クリンナップフェイズ中:1点 | ・自身に付与されたスキル効果を1つ選択し、無効化する。<br>・この効果は戦闘中の場合、任意のタイミングで実行可能。<br>※自身の判定に影響するスキルを無効化する場合、その判定を行う前にレジストを宣言しなければならない。 |
| **宝具レジスト** | クリンナップフェイズ中:3点 | ・自身に付与された宝具効果を1つ選択し、無効化する。 |
| **配置転換** | 1点 | ・戦闘中、自身の手番開始時に自身の配置 (前衛・後衛)を変更する。<br>・このアクションは自陣営の手番中最速のタイミングでしか実行できない。<br>※同盟を含め自陣営の誰かが既に行動済みの場合、このアクションは実行不可<br>・このアクションは1戦闘に1回しか実行できない。 |

### ☆追加HP/追加FP
「防御障壁」や「霊基拡張」等、一部の宝具効果によって戦闘中自身のHP/FPに上乗せされる外付けリソース。
*   このリソースは原則としてスキル効果によって付与されることはない。
*   追加HP/追加FPを獲得した際、ダメージやスキル/宝具の使用等によるリソースの減少/消費は追加HP/FPから優先して行う。
*   複数の宝具によって追加HP/追加FPを獲得した場合、リソースを合算するのではなく獲得した宝具ごとに個別で管理する。(例:追加HPa、追加HPbなど)
*   追加HP/FPはいかなる回復手段を以てしても回復することはできず、戦闘終了後に消滅する。

## 探索技能

探索判定の成功率算出に用いる項目。「探索点」を「肉体」「知覚」「教養」という3つの探索技能に振り分けます。
*   各技能最低1点以上の探索点を振り分けなければならず、原則として「0」以下にすることは出来ない。
*   一つの技能に対しサーヴァントは最大9点、マスターは最大で8点まで振ることができる。

## スキル/宝具について

FPを消費して発動するそのキャラクター固有のアクション。消費するFPは効果内容によって決まる。
*   基本的にはルールにあるサンプルデータをベースに作成する。
*   オリジナル効果を作成することも可能だが、オリジナル効果はGMの監修が必須であり、また「キャラクターの初期英雄点を増やす効果」は原則作成できないものとする。(※①)
*   戦闘中、同じスキル/宝具を同じタイミングで複数回発動させ、その効果を累積させることは出来ない。
*   戦闘中、発動タイミングが同じスキルを同時に使用することは出来ない。また、発動タイミングが異なるスキルであっても、実質的な効果の適用タイミングが重複するのであればGM判断で却下する可能性もあるので注意する。
*   発動タイミングが戦闘を含むエンカウントフェイズ中以外のスキルは例外的に複数同時に発動することができる。
*   「宝具+スキル」や「宝具+宝具」等の組み合わせは同じ発動タイミングでも発動可能とする。
*   基本的に「スキル<宝具」であり、宝具より強力なスキルは作成できない。
*   効果内容が競合した場合、スキル同士や宝具同士であるなら効果を打ち消し合い、スキルと宝具であるなら宝具有利に処理する。その他詳細な判断は適宜GMが判断する。
*   前衛から後衛、後衛から前衛などキャラクターの配置を超えてアイテム、スキル/宝具を使用する場合、効果テキスト内に「この効果は後衛から前衛、前衛から後衛へ使用可能」といった文言が必要となる。(※②)

> ※①: 既にサンプルとして用意されているものは除く
> ※②:前衛/後衛については「フィールドと前衛/後衛」に記載

### スキル/宝具の主な発動タイミング

キャラクター作成時 → シナリオ開始時 → シーン開始 → セットアップフェイズ開始時 → エリア移動時 → セットアップフェイズ終了時 → エンカウント/スタンバイフェイズ開始時 → 戦闘開始時 → 先制判定時 → ラウンド開始時 → 手番開始時 (1 自身/対象の〇判定時; 2 与ダメージ計算時; 3 被ダメージ計算時) → 手番終了時 → ラウンド終了時 → 戦闘終了時 → エンカウント/スタンバイフェイズ終了時 → クリンナップフェイズ開始時 → クリンナップフェイズ終了時 → シーン終了時 → シナリオ終了時
*(手番中のタイミングは任意)*

### スキル/宝具の効果時間

| 効果時間 | 概要 |
| :--- | :--- |
| **判定時** | スキル/宝具の使用を宣言した直後の判定や行動に効果が適用され、適用後すぐにに効果が消失する。<br>主に発動タイミングが「〇判定時」となっているものがこれに該当する。 |
| **ラウンド中**| スキル/宝具の使用を宣言した直後から、宣言したラウンドの終了まで効果が持続する。<br>主に「ラウンド開始時」に発動するスキル/宝具が該当する。 |
| **戦闘中** | スキル/宝具の使用を宣言した直後から、その戦闘が終了するまで効果が持続する。<br>主に効果が累積するものや発動の際に「対抗ロール」を要求するスキル/宝具が該当する。 |
| **永続** | スキル/宝具の使用を宣言した直後から対象がレジスト等の特定条件を満たすまでずっと効果が持続する。<br>効果テキスト中に「永続」の文言と「解除条件」が記載されているスキル/宝具が該当する。 |
| **常時** | 一部のクラススキルやクラス特性など、キャラクリエイト時点から効果を発揮し、シナリオ中持続する。<br>クラススキル「騎乗」や「狂化」等がこれに該当する。 |

### スキル/宝具のタイプ

| タイプ | 取得制限 | 概要 |
| :--- | :--- | :--- |
| **補正値変動系** | キャラ1体につき、スキルで2個、宝具で1個まで取得可能。<br>※攻撃判定/防御判定に影響するスキルは1個のみ。<br>※この制限はクラス特性/クラススキルには適用されない | ①対象の補正値を変動させる能力。<br>②原則として判定前に使用する。<br>③1つの判定につき変動域はスキルの場合『±4』以内、宝具の場合『±8』～『±20』の間で調整を行う。<br>④1回の発動で複数回恩恵を受ける効果である場合、合計変動域は半分かそれ以下となる。<br>⑤特定条件でのみ発動する能力の場合、GMの判断で変動域をさらに拡張することも可能。<br>⑥原則として複数のスキル同士の補正値は累積しない。<br>ただし、専用の記載がある場合や「スキル+宝具」の場合は累積する。 |

## オブジェクト

陣地や工房など、特定のスキル/宝具などによって作成され、エリアに設置されるキャラクター以外の存在。
*   共通項目として「Lv.」があり、「Lv.」によってHP/FPが計算される。尚、HP/FPの計算式やLv.アップの条件などはオブジェクトごとに異なる。
*   オブジェクトは戦闘中「前衛」に強制配置され、敵対陣営は攻撃対象としてこのオブジェクトを選択できる。
*   オブジェクトは判定を行うことは出来ず、また攻撃/防御基準値を持たない。
*   オブジェクトは以下の手段で破棄/破壊される。
    1.  クリンナップフェイズ終了時、作成者が破棄を宣言する。
    2.  エンカウントフェイズにおける戦闘の際に、攻撃を受けてHP0以下になる。
    3.  作成者が設置したエリアから離れている間に他の陣営が設置したエリアに移動してきた時、そのエリアにいるオブジェクト作成者以外の陣営全員が破壊に合意する。この時、オブジェクトの残りHPは問わない。
*   ①や②等によってオブジェクトが破棄/破壊された場合、オブジェクトを再設置できるのは次シーン以降となる。
*   ③ などの作成者の意志に関係なくオブジェクトが破壊された場合、翌シーンのセットアップフェイズ開始時に破壊されたことを把握できる。
*   オブジェクトは単一陣営によるものであれば1エリアに複数設置可能(※1)。
*   複数陣営が単一のエリアにオブジェクトを設置することは出来ない。ただし、エリア結合等で既に作成したオブジェクト同士が意図せず1つのエリアに設置される形になった場合は例外的に複数陣営のオブジェクトが共存することがある。(※2)

> ※①: マスターの工房とサーヴァントの陣地など
> ※②: エリア結合の後、既にオブジェクトが設置されていたエリアに対し新たなオブジェクトを設置することは出来ない。
